# Masa-Cudalign-APA


## Donwload 

Última Versão: 

## Compilando o CUDAlign 

```console
cd masa-cudalign-apa/masa-cudalign
./configure 
make 
```

## Compilando o Módulo APA

No diretório masa-cudalign-apa: 

```console
  g++ modulo_apa.cpp -o apa 
```

## Executando o Módulo APA  

```console
  ./apa [options] seq1.fasta seq2.fasta 
```
